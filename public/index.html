Gadi's API Documentation

Welcome to the API documentation page. Here you can find information about the available endpoints and how to use them.

Users API

POST /api/v1/users

This endpoint creates a new user.

POST /api/v1/users/login

This endpoint logs in a user.

GET /api/v1/users

This endpoint returns a list of all users.

GET /api/v1/users/:id?

This endpoint returns a user by their ID.

PUT /api/v1/users/:id?

This endpoint updates a user's information.

DELETE /api/v1/users/:id?

This endpoint deletes a user by their ID.

PATCH /api/v1/users/:id?

This endpoint changes a user's business status.

Enter the Documentation of Users API for requests, responses, and more details

Cards API

POST /api/v1/cards

This endpoint creates a new card.

GET /api/v1/cards

This endpoint returns a list of all cards.

GET /api/v1/cards/:id?

This endpoint returns a card by its ID.

GET /api/v1/cards/my-cards

This endpoint returns all cards created by the authenticated user.

PUT /api/v1/cards/:id?

This endpoint updates a card's information.

DELETE /api/v1/cards/:id?

This endpoint deletes a card by its ID.

PATCH /api/v1/cards/:id?

This endpoint changes a card's business number.

PATCH /api/v1/cards/:id?

This endpoint likes a card by its ID.
